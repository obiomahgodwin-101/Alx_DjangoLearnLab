# Alx_DjangoLearnLab - Advanced Features and Security

## Week 11: Custom User Model and Permissions

This project demonstrates:
- Custom User Model with `date_of_birth` and `profile_photo`.
- Custom User Manager for creating users and superusers.
- Permissions and groups to control access:
  - `can_view`, `can_create`, `can_edit`, `can_delete` for Books.
- Views protected by permission decorators.
- Admin registration for models.
